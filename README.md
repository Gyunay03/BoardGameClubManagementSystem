# BoardGameClubManagementSystem
## Информационна система за управление на клуб по настолни игри
### Информация относно курсовата задача
Проектът е създаден с цел улесняване на работата на служителите,
като са дигитилизарани следните процеси: проверка за наличност на настолни игри, 
информиране на потребителите за предстоящите събития,както и разглеждане на менюто.
Потребителите имат възможността да резервират места, преглеждат настоящите и/или бъдещите събития,
може да си избират и заемат настолни игри,както и да се запишат за дадено събитие.
Информационната система е започната да се реализира с помощта на ASP.NET,HTML и CSS.
В папка docs се намират функционалните изисквания,релационния модел(диаграма),
диаграмата на потоците от данни,таблицата за вземане на решения,
псевдо-кодовете на някои алгоритми,статията и записката.
В папка src се намира изходния код.
